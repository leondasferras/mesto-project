# Проект: Место

Учебный проект Яндекс.Практикума для студентов факультета web-разработки.

## Использованные технологии

-HTML5 CSS3;  
-JavaScript;  
-Адаптивная верстка  

## Функционал проекта
- Добавление и удаление карточек
- Изменение данных пользователя
- Возможность поставить "лайк"
